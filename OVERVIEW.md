## 🎨 Easy Customization

### Change Hero Background Image
```html
File: index.html (line ~140)
<img src="YOUR-IMAGE-URL" alt="MeowCraft Server" class="hero-bg-image">
```
*Note: Use a high-resolution screenshot (1920x1080 or higher) for best results*

### Change Your Server IP
```html
File: index.html (line ~300)
<code id="server-ip">YOUR-SERVER-IP.com</code>
```

### Update Colors
```css
File: styles.css (line 2-8)
--primary-color: #ff6b35;
--secondary-color: #f7931e;
```

### Add Download Links
```javascript
File: script.js (line ~300)
function downloadLauncher(platform) {
    window.location.href = 'your-download-url';
}
```
